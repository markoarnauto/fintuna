fintuna.FinStudy

class fintuna.FinStudy(Model: Type[ModelBase], data: DataFrame, data_specs: dict = {}, split_specs: dict = {}, name: str = 'finstudy')

Bases: object

A finstudy corresponds to a financial optimization task. A Model is optimized based on the given data. It provides interfaces to get the out-of-sample performance and let’s you fine-tune a model for production.

    ’train_until’ date string, int, float: end of train data, defaults to 60% of tune data
